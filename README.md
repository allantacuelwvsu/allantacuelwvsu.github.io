# allantacuelwvsu.github.io

Personal portfolio website for Allan Tacuel, built as a static GitHub Pages site.

## Structure

- `index.html` - landing page and featured work
- `about.html` - profile, skills, timeline, and interests
- `projects/index.html` - mixed project and creative work directory
- `resume.html` - resume download/viewer placeholder
- `contact.html` - email and profile links
- `css/` - design tokens, components, and page styles
- `js/main.js` - mobile navigation and theme toggle
